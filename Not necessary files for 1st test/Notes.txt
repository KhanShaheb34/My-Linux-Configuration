This guide is for Arch linux/arch based distribution:

	Some Basics of arch:
		
		1.To completely remove all the dependencies with that packages use -Rcns ex:
			yay -Rcns vlc
		2.To find out all the unnecessary packages (packages that donâ€™t depend on other packages):
			yay -Qdtq
			
		And to remove all of them:
			yay -R (yay -Qdtq)

	0.Install necessary softwares
		In manjaro first do this:
			0.yay -Rcns manjaro-hello cheese firefox
		0.sudo pacman -Syyu
		1.sudo pacman -S yay
		2.yay -S neofetch google-chrome brave vlc intellij code qbittorrent xdman handbrake fish sublime-text gimp libreoffice-still mlocate timeshift
			(Add tlp and it's start command for laptop)
			add command of ./install.sh (Embed it into myScript.sh) to unzip and copy existing sublime config file to the desired location.
		[I do not use fish anymore] 3.ohmyfish (curl -L https://get.oh-my.fish | fish) stopped that welcome message of fish (set fish_greeting)		
		
	1.NVIDIA graphics driver:

		4.sudo mhwd -a pci nonfree 0300

	2.Intializing Virtualization:

		5.yay -S virt-manager qemu vde2 ebtables dnsmasq bridge-utils openbsd-netcat
		6.sudo systemctl enable libvirtd.service
		7.sudo systemctl start libvirtd.service
		
		For Setting up the network default:
		
			1.sudo virsh net-list --all
			
			if nothing shows up then->
			2.Create a file called default.xml then write this---
				
				<network>
				  <name>default</name>
				  <uuid>9a05da11-e96b-47f3-8253-a3a482e445f5</uuid>
				  <forward mode='nat'/>
				  <bridge name='virbr0' stp='on' delay='0'/>
				  <mac address='52:54:00:0a:cd:21'/>
				  <ip address='192.168.122.1' netmask='255.255.255.0'>
					<dhcp>
					  <range start='192.168.122.2' end='192.168.122.254'/>
					</dhcp>
				  </ip>
				</network>
			
			3.sudo virsh net-define --file default.xml
			4.sudo virsh net-start default
			5.sudo virsh net-autostart --network default
		
		After everything run: yay -Rcns $(yay -Qtdq)
