Basics of Linux:
	1.Use lf instead of ls for seeing files like a file manager in terminal...

	2.For installing zsh:
		
		yay -S zsh
		
		oh-my-zsh: sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
		
		autosuggestions: cd ~/.oh-my-zsh/custom/plugins/ && git clone https://github.com/zsh-users/zsh-autosuggestions
		
		syntax-highlighting: git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
		
		Replace the .zshrc file with the customized one which is with this file
		
	3.Youtube-dl

		youtube-dl --write-auto-sub --recode-video mp4 youtube.url...
		
		To download songs from youtube:
		
			youtube-dl -ciw --extract-audio --audio-format mp3 youtube.url...
			
		(USE -ciw FOR ANY PLAYLIST)
		
	4.Customizing the terminal:
		
		Themes for gnome-terminal: bash -c "$(wget -qO- https://git.io/vQgMr)"
								   One dark theme: 122
								   
	5.To uncompress a tar.gz file:
		
		tar xvzf file.tar.gz
		
	6.To see the size of a directory:
	
		du -sh /mnt/16E8DB8EE8DB6A8F
		(Ex: It will show me the used size of that disk, I used 774G of 1000G at that time....)
